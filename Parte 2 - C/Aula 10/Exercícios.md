# 1️⃣ Mensagem Personalizada
## Descrição:
Escreva um programa que lê o nome e a idade do usuário e imprima na tela uma mensagem personalizada: *“Olá, [nome]! Você tem [idade] anos.”*

## Entrada:
```
Qual seu nome ? Marcos
Qual sua idade ? 18
```
## Saída:
```
Olá, Marcos! Você tem 18 anos.
```

# 2️⃣ Tamanho da String
## Descrição:
Escreva um programa que leia uma string e informe o tamanho dela (sem contar o \0).

## Entrada:
```
Digite uma palavra: paralelepipedo
```
## Saída:
```
A palavra digitada tem 14 letras.
```

# 3️⃣ Vogais e Consoantes
## Descrição:
Escreva um programa que leia uma string e informe quantas vogais e quantas consoantes ela tem.

## Entrada:
```
Digite uma palavra: paralelepipedo
```
## Saída:
```
A palavra digitada tem:
- 7 vogais
- 7 consoantes
```

# 4️⃣ Várias Operações com String
## Descrição:
Crie um programa que:
- Peça ao usuário que insira seu primeiro nome e sobrenome.
- Use `strcat` para concatenar o nome completo.
- Copie o nome completo para outra variável usando `strcpy`.
- Utilize `strlen` para contar e exibir o número total de caracteres do nome completo.

## Entrada:
```
Nome: Ana
Sobrenome: Oliveira
```
## Saída:
```
Nome completo: Ana Oliveira  
O nome completo tem 13 caracteres.
```
