# 1️⃣ Soma de uma Matriz
## Descrição:
Escreva um programa que retorna a soma dos elementos de uma matriz que possui 9 elementos.

## Exemplo de saída esperada:

### Entrada:
```
6 3 3
7 8 1
9 3 10
```
### Saída:
```
50
```

# 2️⃣ Maior e Menor
## Descrição:
Escreva um programa  que retorna o menor e o maior valor de uma matriz que possui 9 elementos.

## Exemplo de saída esperada:

### Entrada:
```
6 3 3
7 8 1
9 3 10
```
### Saída:
```
Maior: 10
Menor: 1
```

---

# 3️⃣ Soma: Linha e Coluna 
## Descrição:
Escrever um programa que leia uma matriz 3x3 e calcule a soma de cada linha e coluna.

## Exemplo de saída esperada:

### Entrada:
```
6 3 3
7 8 1
9 3 10
```
### Saída:
```
Linha 0: 12
Linha 1: 16
Linha 2: 22
Coluna 0: 22
Coluna 1: 14
Coluna 2: 14
```

---

# 4️⃣ Posição de um Elemento
## Descrição:
Leia uma matriz M × N e um número X. Exiba a posição (linha, coluna) de todas as ocorrências de X. Se o elemento não estiver na matriz imprima `"O elemento não foi encontrado"`.

## Exemplo de saída esperada:

### Entrada:
```
6 3 3 4
7 8 1 9
9 3 7 5
7 
```
### Saída:
```
O 7 aparece nas posições:
(0, 1)
(2, 2)
```
