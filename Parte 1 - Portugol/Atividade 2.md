## 1️⃣ Analisando o CRE

### 📖 Descrição:
Fernanda está organizando um projeto de extensão e precisa selecionar alunos com base no Coeficiente de Rendimento Escolar (CRE). Escreva um programa que permita a ela inserir a matrícula e o CRE de vários candidatos e, ao final, exiba a matrícula do aluno com o menor CRE e a média dos CREs de todos os inscritos.

### 📌 Entrada:
- A **matrícula** (cadeia) e o **CRE** (real) de cada candidato.
- A entrada de dados deve parar quando for informada a matrícula **999.**

### 📌 Saída:
- A matrícula correpondente do menor CRE, seguida pela média de CREs.

### 📌 Exemplo:
#### 📥 Entrada:
```
124  
7.2  
805  
8.4  
902  
6  
999  
```

#### 📤Saída:
```
902  
7.20  
```

## 2️⃣ Árvore de Natal

### 📖 Descrição:

Robson e seus 20 colegas vão montar uma árvore de Natal na empresa e dividir os custos, ela possui três tipos efeite.

### 📌 Entrada:

- O valor da árvore de Natal;
- Para cada um dos três tipos de enfeites: a quantidade comprada (inteiro) e o preço unitário (real).

### 📌 Saída:
- O total gasto com a árvore e os enfeites;
- O valor que cada funcionário deve pagar.

### 📌 Exemplo:
#### 📥 Entrada:
```
300  
10   
6  
5  
8  
5  
4  
```

#### 📤Saída:
```
420.00  
20.00  
```

## 3️⃣ Descontos Black Friday

### 📖 Descrição:

Genésio quer verificar se os descontos da Black Friday são reais. Ele acompanhou 5 produtos em uma loja online, anotando o preço original e o preço promocional para conferir se o abatimento anunciado de 20% foi aplicado. Escreva um programa que receba como entrada, para cada um dos 5 produtos.

### 📌 Entrada:

- O preço original (real);
- O preço na Black Friday (real).

### 📌 Saída:
- A quantidade de produtos que tiveram um desconto de pelo menos 20%.

### 📌 Exemplo:
#### 📥 Entrada:
```
20  
16  
35.40  
28.20  
50  
43  
100  
80  
25  
21.20   
```

#### 📤Saída:
```
3 
```

## 4️⃣ Entregando correspondências

### 📖 Descrição:

Luíz Carlos é um carteiro dedicado e recebeu a meta de entregar pelo menos 100 correspondências por dia durante a alta demanda do Natal.

### 📌 Entrada:

- A quantidade de correspondências entregues por ele em cada um dos sete dias da semana.

### 📌 Saída:
- Em quantos dias ele atingiu ou superou a meta de 100 entregas.
- A média diária de correspondências entregues na semana

### 📌 Exemplo:
#### 📥 Entrada:
```
100  
120  
140  
80  
105  
130  
75  
```

#### 📤Saída:
```
5  
107  
```

## 5️⃣ Quantidade de dígitos 

### 📖 Descrição:

Crie um programa que tenha uma única função, além da principal, que receberá como parâmetro um natural n (0<=n<=2^30) e devolverá a quantidade de dígitos de n. O programa exibirá o retorno da função.

### 📌 Entrada:
- Um número inteiro

### 📌 Saída:
- A quantidade de digitos do número


### 📌 Exemplo:
#### 📥 Entrada:
```
81726354
```

#### 📤Saída:
```
8
```
